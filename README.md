Design assets, sprites, and production resources for the Rolling Candy (formerly Falling Candy) iOS game.

## Directory Structure

| Directory | Contents |
|-----------|----------|
| `sprites/` | Game sprite sheets and individual sprite assets |
| `icons/` | App icons for all required sizes |
| `screenshots/` | App Store screenshots (iPhone, iPad) |
| `marketing/` | Promotional graphics and marketing materials |
| `fonts/` | Custom fonts used in-game |
| `particles/` | Particle effect definitions |
| `physics/` | Physics body shape definitions |
| `shaders/` | Fragment and vertex shaders |
| `preload/` | Assets loaded at game startup |
| `level-maker/` | Level editor sprites and tools |
| `archive/` | Historical assets, earlier versions, and artist deliverables |

### Archive

| Subdirectory | Contents |
|--------------|----------|
| `archive/source-v1/` | Original Objective-C game source (v1) |
| `archive/assets-v2/`, `archive/assets-v3/` | Earlier asset revisions |
| `archive/artist-deliverables/` | Raw files from artists (Davis, Ivars) |
| `archive/compiled/` | Release changelog |
| `archive/sketches/` | Concept sketches and early designs |
| `archive/old-*/` | Superseded icons, screenshots, font, game objects, levels menu |
| `archive/themes/` | Theme-related assets |
